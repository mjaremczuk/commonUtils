# commonUtils
common utils classes

### Maven Dependency for Consumers
Add to your `pom.xml`
````xml
<dependency>
  <groupId>com.github.mjaremczuk</groupId>
  <artifactId>common-utils</artifactId>
  <version>0.0.2</version>
  <type>pom</type>
</dependency>
````

Gradle
````xml
compile 'com.github.mjaremczuk:common-utils:0.0.2'
````
